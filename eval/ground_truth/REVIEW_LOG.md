# Ground Truth Review Log

This document tracks human review of AI-pre-labeled ground truth JSONs.

**Process:**
1. Claude generates ground truth JSON from CV file
2. Elena opens CV file (DOCX/PDF/TXT) alongside the JSON
3. Elena verifies each field against the CV content
4. Elena documents any changes she made below
5. Eval set is considered hand-validated only when all 20 entries are reviewed
